# SafeRoute.AI - Complete Real-Time Route Safety Scoring System

A real-time route safety scoring system that provides pedestrians and vehicle drivers with safe route recommendations based on various safety factors and real-time data.

## 🚀 Project Commands

### Development

```bash
# Install all dependencies
npm install

# Start frontend development server only
npm run dev:frontend

# Start backend (Convex) development server only
npm run dev:backend

# Start both frontend and backend in parallel (recommended)
npm run dev
```

### Building & Production

```bash
# Build the project for production
npm build

# Run TypeScript and lint checks
npm run lint
```

### Backend Setup

```bash
# Setup Convex (initialize backend)
npm run setup
```

## 📋 Development Environment

- **Frontend**: React 19 with TypeScript, Vite, and Tailwind CSS
- **Backend**: Convex serverless platform
- **Port**: Frontend runs on `http://localhost:5173/` (or next available port)
- **Backend URL**: `http://127.0.0.1:3210/` (configured in `.env.local`)

## 🔧 Quick Start

1. Clone the repository
2. Run `npm install` to install dependencies
3. Run `npm run dev` to start both frontend and backend
4. Open `http://localhost:5173/` in your browser
5. Sign up with email/password or continue as guest
